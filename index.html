For...In vs For...Of

An interactive visualizer to see the difference.

for...in

Iterates over the enumerable property names (keys) of an object.

const userProfile = {
  username: 'Alex',
  level: 12,
  is_active: true
};
for (const key in userProfile) { ... }

for...of

Iterates over the values of an iterable object (like an Array).

const userBadges = [
  'Gold',
  'First-Post',
  'Top-Commenter'
];
for (const badge of userBadges) { ... }
