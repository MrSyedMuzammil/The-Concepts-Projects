Break vs. Continue

An interactive visualizer for loop control.

break

Terminates the entire loop permanently when a condition is met.

for (let i = 1; i <= 10; i++) {
  if (i === 5) {
    break; // Stop everything
  }
  console.log(i);
}

continue

Skips the rest of the current iteration and proceeds to the next one.

for (let i = 1; i <= 10; i++) {
  if (i === 5) {
    continue; // Skip this one
  }
  console.log(i);
}
